# Command: explain

Use for architecture or code walkthroughs.

Format:
- GOAL: restate request and stack (Python/Node/C#, React Native, Tailwind/shadcn).
- STRUCTURE: components/modules, APIs, and data flow.
- TESTS: what validates this behavior.
- OBSERVABILITY: logs/metrics/dashboards relevant.
- REFLECT: where a human should verify details.
